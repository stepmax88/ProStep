🛠️ tmux-ps (ProStep Session Manager)

Professional tool for automatic saving and restoring tmux sessions with SSH support.

Автор: Maxim Stepanuk | Компания: ProStep Development

📝 Описание

tmux-ps — это системная утилита для обеспечения непрерывности работы. Она автоматически сохраняет окна, панели и SSH-соединения, позволяя восстановить окружение после перезагрузки или разрыва связи.

🚀 Особенности

  • ⏰ Auto-save: Каждые 15 минут через crontab.
  • 🔄 Auto-restore: При старте системы через systemd.
  • 🛡️ SSH-Persistence: Восстановление активных SSH-клиентов.

🔧 Команды

КомандаОписание
tmux-ps saveСохранить текущее состояние сессий
tmux-ps restoreВосстановить из последнего дампа
tmux-ps lsСписок сессий с деталями
tmux-ps hwidID оборудования для лицензии

📦 Установка (YUM/DNF)

sudo tee /etc/yum.repos.d/prostep.repo <<EOF
[prostep]
name=ProStep Repository
baseurl=https://repo.prostep.com.ua/tmux-ps/repo/
enabled=1
gpgcheck=1
gpgkey=https://repo.prostep.com.ua/tmux-ps/repo/RPM-GPG-KEY-ProStep
EOF

sudo dnf install tmux-ps -y
